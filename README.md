# Flash Cards

A lightweight, single-file flashcard web app. No dependencies, no build step — open the HTML file in a browser or drop it onto a static host like Netlify.

---

## How it works

All cards live in a JavaScript array inside `ml-flashcards.html`. Click a card to flip it, use the buttons to navigate, and shuffle whenever you want a fresh order.

See [how-to-add-cards.md](how-to-add-cards.md) for instructions on adding new cards.

---

## Deployment

Drop `ml-flashcards.html` onto [Netlify Drop](https://app.netlify.com/drop) to publish it instantly. No account required for one-off deploys.
